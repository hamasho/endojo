About Me and This Site

Hello, I'm Shinsuke Hamasho, a Japanese learning English and the developer of this site.

This site is first created to improve my English skills. I have learned English for several years and found that there are three exercises which improved my English skills dramatically. It's the following exercises;

  • Vocabulary exercise,
  • Listening exercise and
  • Transcription exercise.
So I have developed this web site to help me improve my English skills throug these exercises. And then I realized that this site can help all people who learn English.

I'm trying to make this web site better. But because I'm the only one developer and I don't have enough time and money, there may be some bugs and performance issues. If you find such, please contact me.

I'm happy if you play these exercises and improve your skills. Enjoy!
